Following are the funtions included with the file:

  • int main():
    This function initializes the GLUT, creates window and calls the event handling and the display functions. This can be briefly summarized as :

    •       int main(int argc, char *argv[]){

           //initializes GLUT and create window

           //register display call back

           //callback for interaction with keyboard

           //callback for interaction with mouseclick

           //enter event processing cycle

         }

  • void mouse(int button, int state, int x, int y):
    Mouse call back function. Records the position and button when the mouse is clicked to implement the corresponding drawing functions
  • void keyboard( unsigned char key, int x, int y ):
    Key board call back function. Records the key pressed and teh position of mouse when the key is pressed. We have used "switch - case" method to do the required action when the corresponding key is pressed
  • std::string read_filename():
    Used to read the filename that has to be loaded from the user input
  • color_t read_color():
    Used to read the normalised set of RGB from the user input
  • void reshape( int w, int h ):
    Resizes the OpenGL Window. Uses glOrtho to scale the existing matrix of pixels
  • void display( void ):
    Draws the canvas onto the screen
  • void wrapper_set_bg(color_t& bg_color_):
    Sets the background color to black
  • canvas_t canvas(color_t(1,1,1), color_t(0,0,0), point_t(win_width, win_height)):
    Initializes canvas
