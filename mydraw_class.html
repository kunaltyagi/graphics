Following the the classes and the functions implemented in them:

  1. color_t:
    Private variables are red, blue and green components. Consists of the following member functions:
    • color_t(): initializes an element of class and sets R, G B components to zero
    • color_t(const float r_, const float g_, const float b_): Defines a element of type color_t and sets the private variables _r, _g, _b to their corresponding values
    • void set(const float r_, const float g_, const float b_): Sets the private variables _r, _g, _b of the element to the given values
    • bool operator ==(const color_t &b) const: Compare the color of two entities
    • float R(void): Returns the normalized red component
    • float G(void): Returns the normalized green component
    • float B(void): Returns the normalized blue component
  2. pen_t:
    Private variables are current thinkness, current foreground and background color and current mode. Consists of following functions:
    • pen_t(): Initialize an element of class with thickness 1.0, color = white and with mode draw
    • pen_t(int t_, color_t fg_color_, color_t bg_color, mode mode_): Defines an element of class and sets the pen thickness, foreground and background color and the mode
    • set_fg_color(color_t color_): Sets foreground color
    • color_t get_fg_color(void): Returns the current Foreground color
    • set_bg_color(color_t color_): Sets foreground color
    • color_t get_bg_color(void): Returns the current Foreground color
    • color_t get_color(): Get current color of the pen i.e return the foreground color if mode = Draw and background color if mode = Erase
    • float get_width(void): Returns the width of the pen
    • float set_width(float t_): Sets the width of pen to the given number
    • void set_mode(mode mode_): Sets mode of the pen to given value
    • mode get_mode(void): Returns the mode value
    • void set(int t_, color_t fg_color_, color_t bg_color_, mode mode_): Sets the pen thickness, foreground and background color and the mode
  3. point_t
    private variables are x and y coordinates. member functions are:
    • point_t(): Initialize the element of class at origin
    • point_t(int x_, int y_): Defines a element of class and sets it to the given x_ and y_ coordinates
    • int X(void): Returns X value
    • int Y(void): Returns Y value
  4. void X(int x_): Sets X coordinate of given point to x_
  5. void Y(int y_): Sets Y component of given system to y_
  6. void set(int x_, int y_): Sets X and Y components tot he given coordinates
  7. void draw(color_t* color_, canvas_t* canvas_): Calls the edit_pixel function of canvas_
  8. fill_t:
    Used for handling the color to be used for filling. This color is private. Member functions are:
    • fill_t(): Define an element and initialize the current fill color to white
      • ,
    • fill_t(color_t color_): Sets the current fill color to the given value
    • void draw(color_t* color_, point_t* point_, canvas_t* canvas_): Uses floodfill algorithm to color region
    • void draw(color_t* fill_color_, color_t* edge_color_, canvas_t* canvas_): Uses Scanfill algorithm to color region
    • void set_color(color_t color_): Sets the color of element to given value
  9. object_t:
    Maintains the list of all lines and triangles to be drawn. Private variables are array of vertices and its length, _pen, and the point about which scanfill should start. if fill is not implmented this value is (-1,-1)
    • object_t(point_t* point_, int n_, pen_t pen_): Initializes an object with n vertices, whose coordinates are given by point_. Also sets the parameters to that of the pen
    • void set(point_t* point_, int n_, pen_t pen_): Sets an already initilized element to the given values
    • virtual void draw(canvas_t* canvas_) = 0:
    • virtual void fill(point_t* point_, canvas_t* canvas_)
  10. line_t:
    Stores the paramters of the lines drawn. Member functions are:
    • line_t(): Initializes an object.
    • line_t(point_t* point_, pen_t pen_): Calls the object_t(vertice_, 2, pen_) to initialize an object of length 2 with given vertices and pen parameters
    • line_t(point_t start_, point_t end_, pen_t pen_):Initializes and sets the start and the endpoints of line and pen parameters
    • void set(point_t start_, point_t end, pen_t pen_): Sets the start and the endpoints of line and pen parameters
    • void set(point_t* point_, pen_t pen_): Calls object::set to set the vertices.
      • ,
    • void draw(canvas_t* canvas_): Applies modified Bressenhem's algorithm(in all 4 quadrants and no float) and then calls canvas::draw on each pixel
  11. traingle_t():
    Stores parameters of a triangle.
    • triangle_t(): Initilizes an object
    • triangle_t(point_t* vertice_, pen_t pen_): Calls object_t(vertice_, 3, pen_) to initilizes an object of length 3 and set its coordinates and pen parameters as required
    • void set(point_t* vertice_, pen_t pen_): Calls object::set to set the 3 vertices
    • void draw(canvas_t* canvas_): Calls line::draw on each of the three lines to draw to boundaries. If fill_center is not -1, then traingle::fill is called.
    • void fill(point_t* point_, canvas_t* canvas_): Implements the flood fill algorithm
  12. drawing_t :
    It stores all the lines and traingles to be drawn, their color etc which are used to color the pixels. The data of objects and their parameters are stored as private variables. The elements of objects are stored as a vector
    • drawing_t(): Initializes the element
    • void add(object_t* object_): Adds the given object to the list of objects that are already stored
    • void clear(): Deletes all the objects
    • void save(std::string file_): Saves the list of objects and their parameters into a text file using ifstream funcions
    • void load(std::string file_, canvas_t* canvas_) : Loads the objects and their parameters from a text file using ofstream functions
    • void fill(point_t* point_, canvas_t* canvas_): Fills the objects with the specified color
  13. canvas_t:
    Consits of a 2D array of pixels that will be rendered to the screen. Pritive variables include a vector to store the color information of each pixel, another vector to store points, mode, fill_color, pen parameters, drawing. Member functions are:
    Private Functions
    • void _left_click(int x_, int y_): Calls function to add point (x_, y_)
    • void _right_click(int x_, int y_):Calls function to remove point (x_, y_)
    • void _remove_point(point_t point_): Pops the point out of the vector _points

    Public Functions
    • canvas_t(): Initializes the Canvas
    • canvas_t(color_t fg_color_, color_t bg_color_, point_t window_): Sets the canvas parameters to the given values
    • void left_click(int x_, int y_): Calls _left_click(x_, _window.Y() - y_)
    • void right_click(int x_, int y_): Calls _left_click(x_, _window.Y() - y_)
    • void set_size(int width_, int height_): Sets the vector _view_port to the required size, and also sets the _window parameters
    • void set_bg(color_t color_): Sets background color
    • void set_pen_color(color_t color_): Sets pen Color
    • void set_fill_color(color_t color_): Sets fill color
    • void set_pen_width(float t_): Sets pen width
    • void set_pen(pen_t pen_): Sets pen parameters
    • void save_pen(): Saves pen parameter
    • void restore_pen(): Restore the saved pen parameter
    • void edit_pixel(point_t* point_, color_t* color_): Edits the vector _view_port at the given pixel
    • color_t get_pixel(point_t* point_): Returns the color of the point_
    • void draw(void): Draws the objects onto the window using glDrawPixels
    • void clear(void): Clears the canvas and colors all pixels to the background color
    • void save(std::string file_): Saves the drawing into a text file.
    • void load(std::string file_): loads Drawing from a text file.
    • void set_mode(Mode mode_): Sets mode of the pen to draw a point line or a rectangle
    • bool is_valid(point_t* point_): Checks if the point lies within the window
    • void fill(point_t* point_);
    • void add_point(point_t point_);
